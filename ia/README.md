# Módulo de IA e SDD

Esta pasta reúne os arquivos relacionados ao uso de Inteligência Artificial e ao processo de Software Driven Development (SDD), mantendo as informações organizadas e separadas do código-fonte da aplicação.

## Objetivo

Utilizar IA de forma segura e organizada para:

- Estudar a arquitetura do projeto.
- Gerar documentação técnica.
- Criar checklists de QA.
- Identificar e analisar bugs.
- Propor melhorias de código.
- Planejar novas funcionalidades.
- Apoiar revisões de código (Code Review).

## Regras

- Nunca salvar chaves de API neste diretório.
- Nunca enviar o arquivo `.env` para o GitHub.
- Nunca armazenar senhas, tokens ou dados reais de produção.
- Toda melhoria deve possuir uma proposta ou documentação antes de ser implementada.
- As sugestões geradas por IA devem ser revisadas antes de serem aplicadas ao projeto.
