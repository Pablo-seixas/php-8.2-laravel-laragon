# Gemini CLI

## Instalação

`'`bash
npm install -g @google/gemini-cli
```

## Variável de ambiente

> **Importante:** nunca salve sua chave da API no GitHub ou em qualquer repositório público.

No PowerShell, defina a variável apenas para a sessão atual:

```powershell
$env:GEMINI_API_KEY="SUA_CHAVE_AQUI"
```

## Boas práticas de uso

Utilize o Gemini para:

- Analisar a arquitetura do projeto.
- Gerar documentação técnica.
- Revisar código-fonte.
- Criar checklists de testes.
- Sugerir melhorias e refatorações.
- Explicar trechos de código.
- Auxiliar na criação de testes unitários.

## Evite enviar

Nunca envie ao Gemini informações sensíveis, como:

- Arquivos `.env`
- Senhas
- Tokens de acesso
- Chaves de API
- Dados reais de clientes
- Informações confidenciais da empresa
