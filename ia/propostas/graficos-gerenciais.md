# Proposta: Gráficos Gerenciais

## Problema

O sistema possui relatórios em tabela, mas ainda não possui gráficos.

## Objetivo

Criar uma tela de gráficos com dados do estoque.

## Escopo

- Produtos por categoria
- Estoque atual por produto
- Entradas por produto
- Saídas por setor
- Saídas por responsável

## Critérios de aceite

- Deve existir botão "Gráficos" no dashboard.
- A tela deve carregar dados reais do banco.
- Não deve alterar produtos, entradas ou saídas.
- Deve ser apenas leitura.
