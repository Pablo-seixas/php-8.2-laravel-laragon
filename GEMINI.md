# GEMINI.md

## Projeto

Sistema de Controle de Estoque desenvolvido em Laravel 11 com PHP 8.2.

## Objetivo

Manter uma arquitetura limpa, organizada e preparada para utilização de IA como apoio ao desenvolvimento.

## Regras Obrigatórias

- Nunca modificar arquivos sem autorização.
- Nunca acessar ou expor o arquivo `.env`.
- Nunca alterar a pasta `vendor`.
- Nunca apagar arquivos existentes.
- Antes de implementar qualquer funcionalidade, analisar a estrutura atual do projeto.
- Explicar os impactos antes de realizar qualquer modificação.
- Sempre seguir os padrões já existentes no sistema.

## Módulos existentes

- Login
- Usuários
- Produtos
- Categorias
- Entradas
- Saídas
- Relatórios
- Central Analítica
- Dashboard
- Logs
- Acessibilidade
- VLibras

## Tecnologias

- PHP 8.2
- Laravel 11
- Bootstrap 5
- JavaScript
- SQLite

## Padrão de desenvolvimento

- Sempre propor melhorias que preservem as funcionalidades existentes.
- Priorizar código limpo, reutilizável e de fácil manutenção.
- Respeitar a arquitetura atual do projeto antes de sugerir alterações.
- Documentar mudanças importantes sempre que necessário.
